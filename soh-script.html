Scions of Hourglass Script Translator

This tool allows you to translate:
  • English text to SoH script by typing into the text box below
  • SoH script to English text by pressing the different symbols
Additionally, although not all letters have known symbols, this tool provides a speculated version.
You can toggle this with the checkbox below and any letters that are speculative are marked with an asterisk.
